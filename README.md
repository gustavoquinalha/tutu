# Vue default

## Clone the project

``` bash
$ git clone https://github.com/gustavoquinalha/vue-default.git my-app
```
or

``` bash
$ npm install vue-default
```

## Install dependencies
``` bash
$ npm install
```

## Serve with hot reload at localhost:8080
``` bash
$ npm start
```

## Build for production
``` bash
$ npm run build
```
